## Зависимости
Создание виртуального окружения
```commandline
python -m venv .venv
```
Активация виртуального окружения\
Ubuntu
```commandline
source .venv/bin/activate
```
Windows
```commandline
.\.venv\Scripts\activate
```
Установка зависимостей
```commandline
pip install -r .\requirements.txt
```

## Запуск
```commandline
python3 .\main.py
```
